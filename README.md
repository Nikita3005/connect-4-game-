# 🎲 Connect 4 Game 🎮  

A fun and interactive **Connect 4** game built with **Python & Pygame**! 🏆  

---

## 🚀 Features  
✅ Beautiful and responsive game window  
✅ Two-player mode with name input  
✅ Smooth falling pieces animation  
✅ Light blue & white theme for a fresh look
